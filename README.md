# Бушуев Никита БАС-1 Лабораторная №1

## Задача 1

### Текст задачи

Создать список из n одинаковых элементов (элемент вводится) 

### Алгоритм решения

1\. Запрашиваем у пользователя количество элементов n

2\. Проверяем корректность ввода

3\. Используем генератор списка `\[for i in 1..n do yield a]` для создания n одинаковых элементов

4\. Выводим полученный список

## Задача 2

### Текст задачи

Найти произведение цифр натурального числа.

### Алгоритм решения

1\. Запросить у пользователя натуральное число N

2\. Проверить корректность ввода

3\. Определить рекурсивную функцию вычисления произведения цифр:

&nbsp;  Если: число равно 0, вернуть 1 (нейтральный множитель)

&nbsp;  Иначе:

&nbsp;    Взять последнюю цифру числа (остаток от деления на 10)

&nbsp;    Умножить ее на результат рекурсивного вызова для числа без последней цифры (целочисленное деление на 10)

4\. Вызвать функцию для числа N и вывести результат

## Задача 3

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/ 

удаление/поиск элемента, сцепка двух списков, получение элемента по номеру). 

### Алгоритм решения

1\. Добавление элемента в начало списка

Принять элемент и список

Поместить элемент перед всеми элементами списка

Вернуть новый список



2\. Добавление элемента в конец списка

Принять элемент и список

Создать одноэлементный список из элемента

Соединить исходный список с одноэлементным

Вернуть новый список



3\. Удаление первого вхождения элемента

Принять искомый элемент и список

Найти индекс первого вхождения элемента

Если элемент найден:

&nbsp; Взять часть списка до найденного индекса

&nbsp; Взять часть списка после найденного индекса

&nbsp; Соединить эти две части

Если элемент не найден, вернуть исходный список без изменений



4\. Удаление элемента по индексу

Принять индекс и список

Проверить, находится ли индекс в допустимых пределах (от 0 до длины списка - 1)

Если индекс корректен:

&nbsp; Взять часть списка до указанного индекса

&nbsp; Взять часть списка после указанного индекса

&nbsp; Соединить эти две части

Если индекс некорректен, вернуть исходный список без изменений



5\. Проверка наличия элемента в списке

Принять искомый элемент и список

Пройти по всем элементам списка

Если найден элемент, равный искомому, вернуть true

Если список закончился и элемент не найден, вернуть false



6\. Поиск индекса элемента

Принять искомый элемент и список

Объявить рекурсивную функцию с параметром-счетчиком индекса

Рекурсивно обходить список:

&nbsp; Если список пуст: вернуть -1 (элемент не найден)

&nbsp; Если первый элемент равен искомому: вернуть текущий индекс

&nbsp; Иначе: рекурсивно вызвать функцию для хвоста списка, увеличив индекс на 1

Вернуть найденный индекс или -1



7\. Сцепка двух списков

Принять два списка

Последовательно соединить элементы первого и второго списка

Вернуть новый объединенный список



8\. Получение элемента по индексу

Принять индекс и список

Проверить, находится ли индекс в допустимых пределах

Если индекс корректен → вернуть Some(значение)

Если индекс некорректен → вернуть None



9\. Тестирование всех операций

Создать тестовый список \[1; 2; 3; 4; 5]

Последовательно применить каждую операцию

Вывести результаты на экран для проверки корректности работы
